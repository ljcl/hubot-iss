# hubot-iss
[![NPM version][npm-image]][npm-url]
> Ask your hubot things about the current state of the ISS!

```bash
npm install hubot-iss
```

```
hubot> hubot where is the iss now?
hubot> hubot when does the iss fly over Sydney?
```
